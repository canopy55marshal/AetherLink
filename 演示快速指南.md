## 🚀 快速开始演示

### 方式一：使用演示脚本（推荐）

运行快速演示脚本，自动生成演示数据：

```bash
/workspace/projects/scripts/demo-task.sh
```

**功能**：
- ✅ 提供 3 个预设演示任务
- ✅ 自动调用 AI 任务分析
- ✅ 显示任务详情和学习步骤
- ✅ 展示知识点关联
- ✅ 生成前端访问链接

### 方式二：手动演示

1. **打开前端页面**：http://localhost:5000

2. **进入任务分析页面**：
   - 点击首页的"任务分析"入口

3. **输入演示任务**：
   ```
   让编程车抓取苹果
   ```

4. **点击"开始分析"**

5. **查看拆解结果**：
   - 任务详情
   - 学习步骤
   - 知识点关联

## ⚠️ 常见问题

### 问题1：页面加载慢

**原因**：LLM 响应需要时间

**解决**：提前预热，运行一次演示脚本

### 问题2：服务未启动

**解决**：
```bash
cd /workspace/projects
bash .cozeproj/scripts/dev_run.sh
```

### 问题3：知识点关联为空

**原因**：知识库中缺少相关内容

**解决**：使用推荐演示任务
