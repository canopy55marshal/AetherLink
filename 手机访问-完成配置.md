### 方法2：使用 ngrok 公网隧道（最简单）

如果服务器 IP 无法访问，使用 ngrok：

```bash
# 1. 安装 ngrok（只需一次）
cd /tmp && wget https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-linux-amd64.zip && unzip ngrok-v3-stable-linux-amd64.zip && sudo mv ngrok /usr/local/bin/

# 2. 创建隧道
ngrok http 5000

# 3. 复制显示的公网地址
# 例如：https://xxxx-xx-xx-xx-xx.ngrok-free.app

# 4. 在手机浏览器中打开该地址
```

**优点**：
- ✅ 无需网络配置
- ✅ 适合远程演示
- ✅ 临时使用最方便

## ⚠️ 注意事项

### 如果手机无法访问服务器 IP

**可能原因**：
1. 防火墙阻止了外部访问
2. 手机和服务器不在同一网络
3. 云服务器安全组未开放端口

**解决方案**：
1. 使用 ngrok 方案（最简单）
2. 配置防火墙开放端口 5000 和 9091
3. 配置云服务器安全组

### 如果前端请求后端失败

**原因**：前端配置的后端地址是 localhost

**解决方案**：运行一键配置脚本
```bash
/workspace/projects/scripts/quick-mobile-setup.sh
```
