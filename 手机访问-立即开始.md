# ✅ 已为你准备好公网隧道工具

## 🎯 最简单的方法 - 使用 localtunnel

localtunnel 已经安装完成！

---

## 📱 立即在手机上访问

### 步骤1：创建公网隧道

运行以下命令：

```bash
/workspace/projects/scripts/create-tunnel.sh
```

或者直接运行：

```bash
lt --port 5000
```

### 步骤2：复制公网地址

命令运行后会显示类似这样的地址：

```
your url is: https://xxxxxxxx.loca.lt
```

### 步骤3：在手机浏览器中打开

复制这个地址，在手机浏览器中打开即可！

---

## 🚀 快速开始

```bash
# 运行这个命令，然后复制显示的 URL
lt --port 5000
```

就这么简单！

---

## 💡 提示

1. **保持隧道运行**：不要关闭终端，隧道会一直运行
2. **每次使用都需要创建**：隧道地址每次都会变化
3. **稳定连接**：localtunnel 比 ngrok 更轻量，适合临时演示

---

## 🔧 如果 localtunnel 不可用

可以使用 ngrok：

```bash
# 安装 ngrok
cd /tmp
curl -s https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-linux-amd64.zip -o ngrok.zip
unzip ngrok.zip
sudo mv ngrok /usr/local/bin/

# 创建隧道
ngrok http 5000
```
