## 📊 系统概览

### 技术栈
- **前端**: Expo 54 + React Native + TypeScript
- **后端**: Express.js + TypeScript
- **数据库**: Supabase (PostgreSQL)
- **实时通讯**: Socket.io
- **LLM**: coze-coding-dev-sdk (豆包模型)
