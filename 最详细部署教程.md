## 📋 开始部署（只需3步）

---

### 第1步：打开GitHub Pages设置（10秒）

**直接点击这个链接：**

👉 **https://github.com/canopy55marshal/AetherLink/settings/pages**

或者：
1. 访问 https://github.com/canopy55marshal/AetherLink
2. 点击仓库顶部的 **"Settings"**（齿轮图标）
3. 在左侧菜单中找到并点击 **"Pages"**

---

### 第2步：配置GitHub Pages（1分钟）

找到 **"Build and deployment"** 部分，按以下配置：

#### 配置详情：

```
┌─────────────────────────────────────────┐
│  Build and deployment                   │
│  ┌───────────────────────────────────┐  │
│  │  Source: [Deploy from a branch ▼]  │  │
│  │                                 │  │
│  │  Branch: [main       ▼]           │  │
│  │          [/ (root)    ▼]          │  │
│  │                                 │  │
│  │  [Save]  [Cancel]               │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

**操作步骤：**

1. 找到 **"Source"** 下拉框
   - 点击下拉框
   - 选择 **"Deploy from a branch"**

2. 找到 **"Branch"** 下拉框
   - 点击下拉框
   - 选择 **`main`**

3. 找到 **"Folder"** 下拉框
   - 点击下拉框
   - 选择 **`/ (root)`**

4. 点击右下角的 **"Save"** 按钮

5. 等待页面显示绿色提示 **"Your GitHub Pages site is currently being built"**

---

### 第3步：等待并访问（2-3分钟）

GitHub Pages会自动构建和部署，等待2-3分钟后：

**访问链接：**
```
https://canopy55marshal.github.io/AetherLink/
```

**应该能看到：**
- 底部导航栏（6个tab）
- 首页内容
- 所有功能模块都可以点击和交互

## 🔧 如果还是404错误

### 解决方法1：等待更长时间（5-10分钟）
GitHub Pages首次部署可能需要更长时间。

### 解决方法2：清除浏览器缓存
- Windows: 按 `Ctrl + Shift + R`
- Mac: 按 `Cmd + Shift + R`

### 解决方法3：检查配置是否正确
1. 访问 https://github.com/canopy55marshal/AetherLink/settings/pages
2. 确认配置：
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
3. 如果不对，重新配置并保存

### 解决方法4：查看部署日志
1. 在 GitHub Pages 设置页面
2. 查看部署状态
3. 如果有错误，根据错误信息修复
