# 🔧 手机无法访问 - 解决方案

## 问题诊断

✅ **服务状态正常**
- 前端服务：http://localhost:5000 ✅
- 后端服务：http://localhost:9091 ✅

⚠️ **可能的问题**
1. 服务器 IP（9.129.127.116）可能是内网地址，手机无法直接访问
2. 防火墙阻止了外部访问
3. 网络配置问题

---

## 🎯 解决方案（按推荐顺序）

### 方案1：使用 ngrok（最简单、最推荐）⭐⭐⭐⭐⭐

**优点**：
- ✅ 无需配置网络
- ✅ 自动生成公网地址
- ✅ 任何地方都能访问

**步骤**：

#### 1. 安装 ngrok（只需一次）

在服务器上运行：

```bash
# 方法A: 使用 wget
cd /tmp
wget https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-linux-amd64.zip
unzip ngrok-v3-stable-linux-amd64.zip
sudo mv ngrok /usr/local/bin/

# 方法B: 使用 curl（如果 wget 失败）
cd /tmp
curl -s https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-linux-amd64.zip -o ngrok.zip
unzip ngrok.zip
sudo mv ngrok /usr/local/bin/
```

#### 2. 创建隧道

```bash
ngrok http 5000
```

**输出示例**：
```
Session Status                Online
Account                      your_account
Forwarding                   https://xxxx-xx-xx-xx-xx.ngrok-free.app -> http://localhost:5000

Connections                  ttl     opn     rt1     rt5     p50     p90
                             0       0       0.00    0.00    0.00    0.00
```

#### 3. 复制公网地址

复制 ngrok 显示的地址，例如：
```
https://xxxx-xx-xx-xx-xx.ngrok-free.app
```

#### 4. 在手机浏览器中打开

直接在手机浏览器中打开这个地址即可！

---

### 方案2：使用 SSH 端口转发

**适用场景**：手机上有 SSH 客户端

**步骤**：

#### iOS（使用 Termius 或 Blink）
```
ssh -L 5000:localhost:5000 -L 9091:localhost:9091 user@9.129.127.116
```

#### Android（使用 Termux）
```bash
pkg install openssh
ssh -L 5000:localhost:5000 -L 9091:localhost:9091 user@9.129.127.116
```

然后在手机浏览器中访问：
```
http://localhost:5000
```

---

### 方案3：配置防火墙

**适用场景**：服务器允许配置防火墙

**步骤**：

#### 1. 开放端口

```bash
# 如果使用 ufw
sudo ufw allow 5000
sudo ufw allow 9091

# 如果使用 firewalld
sudo firewall-cmd --permanent --add-port=5000/tcp
sudo firewall-cmd --permanent --add-port=9091/tcp
sudo firewall-cmd --reload
```

#### 2. 如果是云服务器

需要在云服务器控制台配置安全组：
- 开放端口：5000（TCP）
- 开放端口：9091（TCP）
- 来源：0.0.0.0/0（所有 IP）

#### 3. 然后访问

```
http://9.129.127.116:5000
```

### 方案4：使用本地隧道（localtunnel）

**步骤**：

```bash
# 安装 localtunnel
npm install -g localtunnel

# 创建隧道
lt --port 5000
```

会显示一个公网地址，复制后在手机浏览器中打开。

## 🔍 诊断命令

如果以上方法都不行，运行以下命令诊断：

```bash
# 检查端口是否开放
netstat -tlnp | grep -E ':(5000|9091)'

# 检查防火墙
sudo ufw status
sudo firewall-cmd --list-ports

# 测试本地访问
curl -I http://localhost:5000
curl http://localhost:9091/api/v1/health

# 查看 IP 地址
hostname -I
curl ifconfig.me  # 查看公网 IP
```
