# 📱 手机访问项目 - 快速开始

## 🎯 最简单的方法（推荐）

### 方案一：直接使用服务器 IP 地址

**前提条件**：
- ✅ 服务器允许外部访问
- ✅ 手机可以访问服务器 IP

**步骤**：

1. **获取服务器 IP**
   ```bash
   hostname -I | awk '{print $1}'
   ```
   **当前 IP**：`9.129.127.116`

2. **一键配置**
   ```bash
   /workspace/projects/scripts/quick-mobile-setup.sh
   ```
   这个脚本会：
   - ✅ 自动配置前端使用服务器 IP
   - ✅ 重启前后端服务
   - ✅ 显示手机访问地址

3. **手机访问**
   ```
   http://9.129.127.116:5000
   ```

---

### 方案二：使用 ngrok 公网隧道（最简单）⭐

**优点**：
- ✅ 无需配置网络
- ✅ 自动生成公网地址
- ✅ 适合临时演示

**步骤**：

1. **安装 ngrok**
   ```bash
   cd /tmp
   wget https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-linux-amd64.zip
   unzip ngrok-v3-stable-linux-amd64.zip
   sudo mv ngrok /usr/local/bin/
   ```

2. **创建隧道**
   ```bash
   ngrok http 5000
   ```

3. **复制公网地址**
   ngrok 会显示类似这样的地址：
   ```
   https://xxxx-xx-xx-xx-xx.ngrok-free.app
   ```

4. **手机访问**
   直接在手机浏览器中打开这个地址即可！

## ⚠️ 常见问题

### 问题1：手机无法访问服务器 IP

**解决方案**：使用 ngrok 方案

### 问题2：前端请求后端失败

**原因**：前端配置的后端地址是 localhost

**解决方案**：运行一键配置脚本
```bash
/workspace/projects/scripts/quick-mobile-setup.sh
```

### 问题3：ngrok 隧道不稳定

**原因**：免费版 ngrok 地址会变化

**解决方案**：
- 每次演示时重新创建隧道
- 或使用付费版固定域名
