# AetherLink Demo 访问指南

## 🌟 在线体验 Demo

### 方式1：GitHub Codespaces（推荐，无需安装）

⏱️ **耗时：约2-3分钟**

1. 访问项目仓库：https://github.com/canopy55marshal/AetherLink
2. 点击绿色的 **"Code"** 按钮
3. 选择 **"Codespaces"** 标签
4. 点击 **"Create codespace on main"**
5. 等待环境创建（约1-2分钟）
6. 在终端中依次运行：
   ```bash
   pnpm install
   pnpm dev
   ```
7. 点击浏览器中弹出的链接，即可访问Demo

---

### 方式2：本地运行（需要安装 Node.js）

⏱️ **耗时：约5-10分钟**

#### 环境要求
- Node.js >= 18
- pnpm >= 9.0.0

#### 运行步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/canopy55marshal/AetherLink.git
   cd AetherLink
   ```

2. **安装依赖**
   ```bash
   pnpm install
   ```

3. **启动服务**
   ```bash
   pnpm dev
   ```

4. **访问应用**
   - 前端（Web）：http://localhost:5000
   - 后端API：http://localhost:9091

---

## 📱 移动端访问

### 使用 Expo Go App

1. 在手机上下载 [Expo Go](https://expo.dev/client)（iOS/Android）
2. 扫描终端中显示的二维码
3. 即可在手机上体验完整的移动端应用
