## 📱 演示流程（10-15分钟）

### 第一部分：首页展示（2分钟）

**演示内容**：
1. **欢迎界面**
   - 应用标题和品牌形象
   - 底部 Tab 导航（首页、认知馆、数据制作、爱宠、社区、我的）

2. **核心功能入口**
   - 任务分析入口（AI 任务拆解）
   - 热门任务推荐
   - 快速学习入口

**解说词**：
> "这是一个专注于未来核心资源素养教育的智能学习平台，集成了 AI 任务拆解、知识学习、实战练习等功能。"

### 第二部分：AI 任务拆解（核心功能，5分钟）

**演示内容**：

1. **输入任务**
   - 进入"任务分析"页面
   - 输入示例任务：**"让编程车抓取苹果"**
   - 点击"开始分析"

2. **AI 智能拆解**
   - 展示 AI 分析过程（加载动画）
   - 展示拆解结果：
     - 任务标题：编程车抓取苹果实现
     - 任务描述：通过编程控制移动机器人完成苹果的识别与抓取动作
     - 难度等级：中级
     - 预计时间：240分钟
     - 学习步骤：4个步骤

3. **查看详细步骤**
   - 进入任务详情页
   - 展示每个步骤：
     - 步骤1：认知编程车抓取系统硬件
     - 步骤2：掌握编程车核心编程逻辑
     - 步骤3：编写苹果抓取任务完整代码
     - 步骤4：调试优化抓取任务流程

4. **展示知识点关联** ⭐
   - 点击任一步骤，展开"推荐学习资源"
   - 展示**知识链**：
     - 编程基础入门
     - 机器人编程实战
     - 传感器应用技术
     - 自动化控制原理
   - 展示**知识文章**：
     - 编程入门：从零开始写第一个程序
     - 机器人的"大脑"：控制器编程指南
     - 传感器入门：检测世界的眼睛
     - 自动化控制：让机器为你工作

5. **开始学习**
   - 点击"开始学习"按钮
   - 跳转到第一个知识链或文章

**解说词**：
> "这是我们平台的核心功能 - AI 任务拆解。当用户输入一个学习任务后，AI 会自动分析任务目标，将其拆解为可执行的学习步骤，并为每个步骤推荐最相关的知识内容。这样用户就能从零开始，系统性地完成学习目标。"

### 2. 常见问题处理

**问题1：页面加载慢**
- 原因：LLM 响应需要时间
- 解决：提前预热，提前执行一次任务分析

**问题2：知识点关联为空**
- 原因：知识库中没有相关内容
- 解决：使用准备好的任务列表

**问题3：网络连接问题**
- 原因：后端服务未启动
- 解决：
  ```bash
  cd /workspace/projects/server
  NODE_ENV=development pnpm run dev &
  ```
