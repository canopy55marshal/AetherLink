## 🚀 只需要2步就能看到Demo（2分钟搞定！）

### 第1步：运行部署脚本（1分钟）

在项目根目录运行：

```bash
./scripts/deploy-to-github-pages.sh
```

脚本会自动完成所有操作，你只需要等待。

### 第2步：在GitHub上启用Pages（1分钟）

1. 访问：https://github.com/canopy55marshal/AetherLink/settings/pages
2. 在 **"Source"** 下：
   - Branch 选择：`main`
   - Folder 选择：`/ (root)`
3. 点击 **"Save"**（保存）

---

## 🎉 完成！

等待1-3分钟，然后访问：

```
https://canopy55marshal.github.io/AetherLink/
```

就能看到Demo了！

---

## 📋 如果链接打不开

1. 确认GitHub Pages已启用（参考第2步）
2. 等待1-3分钟，部署需要时间
3. 检查链接是否正确：`https://canopy55marshal.github.io/AetherLink/`
