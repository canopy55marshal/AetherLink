## ⏱️ 只需要3分钟，完成3个步骤

### 第1步：访问GitHub Pages设置（10秒）

点击这个链接直接跳转到设置页面：

👉 **https://github.com/canopy55marshal/AetherLink/settings/pages**

---

### 第2步：配置GitHub Pages（20秒）

在页面上找到 **"Source"** 部分，配置如下：

```
Build and deployment
    │
    └── Source: Deploy from a branch
            │
            ├── Branch: main
            └── Folder: / (root)
```

**具体操作：**
1. 在 **"Source"** 下拉框选择 **"Deploy from a branch"**
2. 在 **"Branch"** 下拉框选择 **`main`**
3. 在 **"Folder"** 下拉框选择 **`/ (root)`**
4. 点击右侧的 **"Save"** 按钮

---

### 第3步：等待并访问（2-3分钟）

GitHub Pages会自动部署，等待2-3分钟后，访问：

```
https://canopy55marshal.github.io/AetherLink/
```

就能看到完整的Demo了！

## 🎯 如果还是404错误

1. **等待更长时间**：GitHub Pages首次部署可能需要3-5分钟
2. **清除浏览器缓存**：按 `Ctrl+Shift+R` 强制刷新
3. **检查配置**：确保Branch是 `main`，Folder是 `/ (root)`
4. **查看部署状态**：
   - 访问：https://github.com/canopy55marshal/AetherLink/settings/pages
   - 查看 **"Build and deployment"** 部分的状态
