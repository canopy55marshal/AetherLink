# 🎯 一键部署到GitHub Pages

## 📱 客户访问Demo的链接

👉 **https://canopy55marshal.github.io/AetherLink/**

---

## 🚀 3步完成部署（超级简单！）

### 第1步：运行部署脚本

在项目根目录运行：

```bash
./scripts/deploy-to-github-pages.sh
```

脚本会自动完成所有操作，你只需要等待。

### 第2步：在GitHub上启用Pages

1. 访问仓库：https://github.com/canopy55marshal/AetherLink
2. 点击 **"Settings"**（设置）
3. 找到左侧菜单的 **"Pages"** 选项
4. 在 **"Source"** 下：
   - Branch 选择：`gh-pages`
   - Folder 选择：`/ (root)`
5. 点击 **"Save"**（保存）

### 第3步：等待部署完成

等待1-3分钟，访问以下链接：

```
https://canopy55marshal.github.io/AetherLink/
```

就能看到Demo了！

---

## 🎉 完成！

现在你可以把链接分享给客户了：

👉 **https://canopy55marshal.github.io/AetherLink/**

客户无需任何配置，直接在浏览器打开就能看到完整的Demo！

---

## 📋 如果链接打不开

1. 确认GitHub Pages已启用（参考第2步）
2. 等待1-3分钟，部署需要时间
3. 检查链接是否正确：`https://canopy55marshal.github.io/AetherLink/`
